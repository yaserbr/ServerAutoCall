AutoCall Setup Guide

Required Permissions

  1. Install AutoCall from the Download button.
  2. Open the AutoCall after installation.
  3. Allow Phone permission.
  4. Allow SMS permission.
  5. Enable Display over other apps / Appear on top.
  6. Set Battery usage to Unrestricted if available.
  7. Return to the AutoCall app and make sure the Device UID is visible.

Enable Accessibility Service

  1. Open Android Settings.
  2. Go to Accessibility.
  3. Select AutoCall Service.
  4. Turn the service ON.
  5. Confirm any Android warning dialogs.
  6. If Android says the service is not working correctly, turn it OFF and ON again.
  7. Return to AutoCall and test Return to AutoCall from the control panel.
